Conectando estudantes e natureza

Um projeto interdisciplinar desenvolvido no Colégio Cívico Militar Santos Dumont, visando integrar as diferentes áreas do conhecimento, promovendo uma educação ambiental, o desenvolvimento de boas práticas, a conexão entre estudantes e natureza, possibilitando valorização e estreitamento dos laços entre o campo e a cidade, por meio da participação ativa dos estudantes em sociedade.

Alguns temas que estão sendo trabalhados

Alimentação Saudável

Educação Ambiental

Desenvolvimento de habilidades sociais e emocionais

Conexão entre campo e cidade

Nosso Projeto

Início

O projeto é de autoria dos professores Cláudio Gabriel, Denilton Gabriel e Lilian Alexandre, que se uniram para proporcionar aos alunos, uma atividade interdisciplinar visando o promover uma construção entre cidadania, meio ambiente e sociedade.

Turma 2025

Este projeto é uma página web que apresenta trabalhos dos meus alunos do Colégio Cívico Militar Santos Dumont 2º ano turma A.

Turma 2025

Este projeto é uma página web que apresenta trabalhos dos meus alunos do Colégio Cívico Militar Santos Dumont 3º ano turma A.

Turma 2025

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção pessoal de atividades de alunos de Ensino Médio. O objetivo é criar um ambiente virtual onde seja possível compartilhar atividades que se destacaram durante o ano letivo de 2025.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site.
